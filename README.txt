(1)使い方

　ChromeブラウザがインストールされているWindows端末にて
　application/Main.javaを実行

(2)仕様についての注意事項

　・「入力値設定」機能は未実装です。
　・歩数･睡眠時間の入力値は設定から変更できます。
　・歩数･睡眠時間の入力中に、動作がストップしてしまうことがあります。
　　設定から「ウェイト」の値を増やしてみてください。
　・chromedriverのバージョンが違うというエラーが出た場合、
　　ブラウザと同じバージョンのchromedriver.exeをダウンロードし、
　　exeフォルダ下のchromedriver.exeを置き換えてください。
　・（既知のバグ）exe化してから実行すると、歯車アイコンが表示されない。
