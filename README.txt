              Pep-Up-Inputter ver.1.1.

(1)使い方

　application/main

(2)動作環境

　Windows10
　Javaの実行環境(Eclipseを想定)
　Chromeブラウザ

(3)仕様についての注意事項

　・「入力値設定」機能は未実装です。
　・歩数は固定で8000歩が入力されます。
　・睡眠時間は固定で7時間が入力されます
　・歩数･睡眠時間の入力中に、動作がストップしてしまうことがあります。
　　設定から「ウェイト」の値を増やしてみてください。
　・chromedriverのバージョンが違うというエラーが出た場合、
　　ブラウザと同じバージョンのchromedriver.exeをダウンロードし、
　　exeフォルダ下のchromedriver.exeを置き換えてください。
　・個人的に作成したアプリなので、テスト密度は低い点をご了承ください。
　・（既知のバグ）exeから実行すると歯車アイコンが表示されない。
